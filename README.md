# Ranch Builder

A MelonLoader mod for Slime Rancher 2 that adds custom building and sandbox tools.

## Features

- **Custom corrals** — functional pens that slimes actually use, including upgrades
- **Houses** — buildable houses with walls and roof, sleep in them
- **Free draw** — draw structures freehand in 3D space
- **Floor builder** — place flat surfaces at any size
- **Polygon tool** — place predefined shapes (walls, platforms, etc.)
- **74 procedural materials** — all tileable, with normal maps and parallax
- **Material painter** — paint any surface with any material
- **Remove tool** — delete placed structures
- **Real economy** — items cost in-game currency, no free spawns
- **Multi-language** — ES / EN / ZH / RU / FR
- **Disk cache** — textures generated once, loaded instantly on subsequent runs

## Installation

1. Install **MelonLoader** on Slime Rancher 2
2. Download the latest `SlimeCorralSpawn.dll` from [Releases](https://github.com/AlkaPrime12/Ranch-Builder/releases)
3. Place the `.dll` in `[Slime Rancher 2 folder]/Mods/`
4. Launch the game — press **F5** to open the build menu
